## ✨ المميزات

✅ **تلخيص ذكي** - يلخص النصوص العربية الطويلة مع الحفاظ على المعنى الأساسي  
✅ **إحصائيات لحظية** - يعرض عدد الكلمات قبل وبعد التلخيص ونسبة التبسيط  
✅ **نسخ بنقرة واحدة** - زر مخصص لنسخ النتيجة بسرعة  
✅ **واجهة عربية** - دعم كامل للغة العربية والكتابة من اليمين لليسار  
✅ **تصميم متجاوب** - يعمل على جميع الأجهزة (جوال، تابلت، كمبيوتر)  
✅ **مجاني للأبد** - بدون أي رسوم أو اشتراكات  
✅ **دعم المجتمع** - زر "اشتر لي قهوة" لدعم استمرارية التطوير
